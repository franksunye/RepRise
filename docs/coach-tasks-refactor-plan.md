# Coach Tasks 页面改动计划

## 当前状态 vs 设计需求

### 当前实现
- ✅ 统计卡片（总数、进行中、待开始、已逾期）
- ✅ 新建任务按钮
- ✅ 按状态分组显示（进行中、待开始、已逾期）
- ✅ 任务卡片显示基本信息
- ✅ 查看详情、编辑按钮
- ✅ 逾期任务特殊样式

### 设计需求
- ❌ 过滤/筛选控件（显示范围、按代表、优先级）
- ❌ 统一的表格/列表视图
- ❌ 标为完成操作
- ❌ 创建时间列
- ❌ 空状态处理
- ❌ 完整的操作列

---

## 必须改动清单

### 【高优先级】

#### 1. 添加过滤/筛选控件
**位置**: 顶部操作栏下方

**需要添加**:
- [ ] 显示范围下拉菜单：所有 / 待处理 / 进行中 / 已完成 / 逾期
- [ ] 按代表下拉菜单：选择特定销售代表
- [ ] 优先级下拉菜单：高 / 中 / 低
- [ ] 搜索框（可选）

**原因**: 设计明确要求，便于教练快速定位和过滤任务

---

#### 2. 改为统一的表格/列表视图
**当前问题**: 按状态分组的卡片视图，不符合设计

**需要改动**:
- [ ] 改为单一的表格或卡片列表
- [ ] 显示所有任务（不按状态分组）
- [ ] 每行显示完整信息

**表格列**:
| 列名 | 说明 |
|------|------|
| 任务标题 | 可点击打开详情 |
| 代表 | 销售代表名字 |
| 类型 | 任务类型 |
| 优先级 | 高/中/低 |
| 状态 | Pending/In Progress/Completed/Overdue |
| 到期时间 | 截止日期 |
| 创建时间 | 任务创建日期 |
| 操作 | 查看/编辑/完成 |

**原因**: 设计明确要求表格形式，便于一览所有任务

---

#### 3. 添加"标为完成"操作
**当前缺少**: 只有查看和编辑

**需要添加**:
- [ ] "标为完成"按钮/操作
- [ ] 点击后更新任务状态为 Completed
- [ ] 可能需要确认对话框

**原因**: 教练需要快速标记任务完成

---

### 【中优先级】

#### 4. 添加空状态处理
**需要添加**:
- [ ] 当没有任务时显示空状态
- [ ] 提示文案：你目前没有任何辅导任务
- [ ] 按钮：创建你的第一个任务
- [ ] 可选说明文案

**原因**: 提升用户体验，引导用户创建任务

---

#### 5. 添加创建时间列
**当前缺少**: 设计要求显示创建时间

**需要添加**:
- [ ] 在表格中添加"创建时间"列
- [ ] 显示任务创建日期

**原因**: 便于教练了解任务的创建时间

---

### 【低优先级】

#### 6. 优化状态可视化
- [ ] 使用更清晰的状态标签
- [ ] 逾期任务的高亮显示（红色）
- [ ] 状态颜色编码

---

#### 7. 添加分页/滚动
- [ ] 如果任务很多，添加分页控件
- [ ] 或实现无限滚动

---

## 实现顺序

1. **第一阶段**: 改为表格/列表视图 + 添加过滤控件
2. **第二阶段**: 添加"标为完成"操作 + 空状态处理
3. **第三阶段**: 优化状态可视化 + 分页

---

## 技术考虑

- 需要添加状态管理（过滤条件）
- 需要实现过滤逻辑
- 可能需要添加新的 UI 组件（Select/Dropdown）
- 需要处理空状态
