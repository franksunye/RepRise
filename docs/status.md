# RepRise 项目状态

## 项目概述

- **项目名称**: RepRise - 管家销售赋能平台
- **开发时间**: 2024-11-17
- **技术栈**: Next.js 14, TypeScript, Tailwind CSS, shadcn/ui
- **开发方式**: 纯前端应用，使用模拟数据（mock data），无后端依赖（可后期接入）

**目标摘要**: 通过 AI 驱动的模拟练习、个性化辅导和数据分析，帮助销售团队提升技能与业务绩效。核心面向角色为销售管家（Rep）与教练（Coach）。
