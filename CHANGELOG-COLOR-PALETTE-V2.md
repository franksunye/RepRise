# 配色方案升级 v2.0 - 变更日志

## 📋 概述

RepRise 配色方案从 v1（橙色为主）升级到 v2（蓝色为主，中性灰，橙色为辅），以提升专业感和用户体验。

## 🎨 核心变更

### 配色原则：60-30-10 规则

- **60% 中性灰** - 背景、卡片、文本、边框
- **30% 蓝色** - 主要交互、链接、导航
- **10% 橙色** - 关键 CTA、重要提示、品牌标识

### 主要颜色变更

| 用途 | v1 颜色 | v2 颜色 | 说明 |
|---|---|---|---|
| 主按钮 | `#F26430` (橙) | `#2563EB` (蓝) | 改用蓝色作为主要交互色 |
| 链接 | `#F26430` (橙) | `#2563EB` (蓝) | 改用蓝色 |
| 关键 CTA | `#F26430` (橙) | `#F26430` (橙) | 保留，但减少使用频率 |
| 页面背景 | `#F9F9F9` | `#FAFAFA` | 更浅、更中性 |
| 卡片背景 | `#EDE8E4` | `#F5F5F5` | 更浅、更中性 |
| 主文字 | `#4A4744` | `#404040` | 更深、对比度更高 |
| 边框 | `#D6D0CA` | `#D4D4D4` | 更中性 |
| 成功色 | `#16A34A` | `#22C55E` | 略微提亮 |

## 🎯 进度条配色策略

### 语义化配色

- **蓝色** - 常规进度条（60-70%）
  - 统计卡片、常规指标
  - 60-89 分的练习成绩

- **绿色** - 成功/优秀进度条（20-30%）
  - 90 分以上的练习
  - 已完成的任务

- **橙色** - 强调/紧急进度条（仅 10%）
  - 紧急任务标记
  - 60 分以下需要改进的练习
  - 关键 CTA 按钮

## 💡 设计理念

v2.0 的设计遵循现代 SaaS 产品的最佳实践：

1. **专业感** - 蓝色是 SaaS 产品的标准选择
2. **可信赖** - 冷色调中性灰提升专业度
3. **清晰的信息层次** - 语义化配色让用户快速理解
4. **减少视觉疲劳** - 橙色仅用于真正需要关注的地方
5. **可访问性** - 所有组合满足 WCAG 2.1 AA 标准
