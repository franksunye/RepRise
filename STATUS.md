# RepRise 项目状态

## 已完成功能

### 管家端
- 首页 - 任务、练习、KPI 概览
- 学习路径 - 4门入职课程（漏水识别、维修流程、报价知识、沟通技巧）
- 模拟练习 - 电话沟通场景
- 练习历史 - 记录和评分
- 内容库 - 话术、模板、清单
- 任务管理 - 状态跟踪
- 通知中心 - 消息提醒
- 我的数据 - KPI、趋势、能力分析

### 教练端
- 首页 - 团队概览
- 管家管理 - 列表和详情
- 辅导任务 - 布置和跟踪
- 团队分析 - 数据统计
- 能力分析 - 雷达图

### 基础功能
- 角色切换（管家/教练）
- 真实头像（DiceBear API）
- 响应式设计
- 模拟数据系统

## 待开发功能

## 技术栈
- Next.js 14 + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts（数据可视化）
- DiceBear（头像）
